# Deenathayalan D — Portfolio

Static portfolio ready for GitHub Pages.

## Publish it

1. Create a GitHub repository and upload all files in this folder.
2. In GitHub, open **Settings → Pages**, choose **Deploy from a branch**, then select `main` and `/ (root)`.
3. In **Settings → Pages → Custom domain**, enter `deenathayalan.dev`.
4. At your domain provider, point the domain to GitHub Pages using the DNS records GitHub shows you.

The included `CNAME` file ensures GitHub Pages uses `deenathayalan.dev` after the domain is verified.
